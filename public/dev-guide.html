Harsh Pandey Portfolio - Development Guide

Opening the Portfolio Website

To view the portfolio website, simply open the index.html file directly in your web browser.

Starting a Local Development Server

If you have Python installed, you can start a simple HTTP server:

Using Python (Python 3):

python -m http.server

Then open http://localhost:8000 in your browser

Using Python (Python 2):

python -m SimpleHTTPServer

Then open http://localhost:8000 in your browser

Using Node.js:

If you have Node.js installed, you can run:

npx http-server

Then open the URL displayed in the terminal (usually http://localhost:8080)

Editing Files

You can edit the portfolio files with any text editor or IDE:

  • index.html - Main HTML structure
  • index.css - Styling and animations
  • script.js - Interactive functionality and animations
